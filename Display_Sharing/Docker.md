# Docker
 - 컨테이너 기반의 오픈소스 가상화 플랫폼
 - 컨테이너와 함께 이미지
   - 이미지 : 컨테이너 실행에 필요한 파일과 설정값등을 포함하고 있는것, 변하지 않는 값(Immutable)[Window 설치 iso 같은?]
 - 도커 이미지는 [Docker hub](https://hub.docker.com/)에 등록하거나 Docker Registry 저장소를 직접만들어 관리 할 수 있다. 누구나 쉽게 이미지를 만들고 배포가 가능
